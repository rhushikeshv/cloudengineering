# Cloud Engineering with Pulumi


````
Install pulumi
Install awscurl

Run the following commands

pulumi up

To test the API run the command

awscurl --service execute-api -X POST -d "hello" https://8q3minevj2.execute-api.us-east-1.amazonaws.com/example/proxy
````
